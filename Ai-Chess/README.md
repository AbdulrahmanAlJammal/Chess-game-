**NOTE** I did not make this game. You can find the original version [here](https://github.com/ankith26/My-PyChess). This is a modified version for a project.

## How to run

- Install **python**
- Install **pygame**
- Clone/Download this repository
- Navigate to directory
- Run **mychess.py**
